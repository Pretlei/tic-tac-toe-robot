# tic-tac-toe-robot
An autonomous Tic Tac Toe robot you can play against! Huge thanks to Maxim Beauchemin, Chris Dietcher, and Andrew Jang for their shared commitment and hard work that made this project possible.

## Project Overview:
An autonomous Tic Tac Toe Robot built using VEX IQ and Tetrix components, as well as 3D printed parts and an aluminum extrusion. Uses a 3-axis motor system with color and distance sensors to interact with the X's and O's. Programmed game logic and robot instructions using C++ and VEX IQ Code.

## Issues with the Tic Tac Toe Robot:
- Motor inaccuracy
- Sensor inaccuracy
- Overheating of VEX IQ motors
- Slight lack of proper building components, and therefore, structural integrity

Unfortunately, time and resources were severe constraints in this project. We made the best of what we had! It was an excellent learning experience.
